# Timeless Energy Principle (TEP)

This repository accompanies the manuscript:

**"The Timeless Energy Principle: An Effective Thermodynamic Framework for Emergent Space-time and Early-Universe Cosmology"**

## Public Datasets

The observational datasets used in this study are publicly available from the following official repositories:

- Planck 2018 Legacy Archive: https://pla.esac.esa.int/
- Pantheon+ SH0ES Data Release: https://github.com/PantheonPlusSH0ES/DataRelease
- DESI Data Release: https://data.desi.lbl.gov/

## Repository Purpose

This repository provides supporting materials associated with this study. The public observational datasets used in the analysis are linked above. Additional computational materials and analysis scripts will be added to this repository as they are prepared for public release.
